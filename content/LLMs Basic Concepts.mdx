### コンテキスト（context）とは

コンテキストとは、ユーザーが何かを尋ねるときに LLM に提供する情報です。

例えば、ユーザーは以下の質問をLLMに聞いた時に、下線付きの単語はユーザーが LLM に与えるコンテキストです。

“この本に対して、<u>100語以内</u>の概要を教えてください。”

そのコンテキストのおかげで、LLM は回答を微調整する方法を知っています。

### コンテキストウィンドウ（context window）とは

コンテキスト ウィンドウはLLM に提供できるコンテキストの最大サイズです。

```
chatGPTのコンテキストの最大サイズは
- chatGPT 3.5：4,096 tokens (approx. 3,000 words)
- chatGPT 4: 8,192 tokens (approx. 6,100 words)
```

### トークン（token）とは

- トークンは普通な単語の小さな部分です。
- 複数のトークンからシーケンスを形成します。
- LLM でのすべてのシーケンスが語彙を構成します。

### プロンプト（prompt）とは

プロンプトは基本的に LLM に与えるコンテキストです。

例えば、以下の三つの質問がchatGPTに聞いた時に、これらの各質問がプロンプトになります。

1. “この本に対して、100語以内の概要を教えてください。”
2. "相対性理論を説明してください。"
3. "あなたは植物学の教授になったかのように、光合成を説明してください。"

したがって、LLMからの応答の品質はプロンプトの品質に依存します。
